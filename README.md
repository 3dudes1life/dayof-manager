# DayOf Manager V0.1

A mobile-first prototype for building and running live events.

## Included

- Multiple event workspace
- Event setup: name, venue, date, hours, manager, accent color
- Schedule builder
- People directory
- Team roles and event code
- CSV schedule import and template
- JSON event export and restore
- Event duplication
- Demo event
- Live Current / Next / After dashboard
- Complete and Stage Ready controls
- Schedule delays
- Day-of schedule
- Check-ins
- Call, text, and email actions
- Issues and quick logging
- Handoff notes
- Offline PWA support

## V0.1 limitation

This version stores events locally in the browser. Multi-device Cloudflare syncing, customer accounts, permissions, Stripe billing, and real invite links are the next product phase.

## Install

Upload every file to the root of a new GitHub repository and enable GitHub Pages from `main`.

Recommended repository name: `dayof-manager`

Open the app and use `••• → Load Demo Event` to see the complete product flow.
